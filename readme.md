# customizable short-cut bluetooth keyboard

build enviroment : PlatformIo

# Requirement
* Visual Studio Code
* PlatfromIO

# setup
* flash program
* edit config.ini file
* connect USB to PC
* format attatched disk with FAT
* copy config.ini file to attatched disk
